

Pre-release of the workshop for Pi and More 10 1/2.

This is only for testing.

To install, open a terminal window, copy the entire block below, paste it into the terminal window and hit enter.


#---------------------------------
release=scratchClient-Tutorials-v0.1-alpha
#THIS IS THE RELEASE NUMBER, e.g. scratchClient-Tutorials-v0.1-alpha
cd ~ # go to the home folder
wget https://github.com/hansdejongehv/scratchClient-Tutorials/archive/$release.tar.gz # download the archive
tar xzf $release.tar.gz # unpack the archive.
cd *$release # change directory to the directory into which into which the archive was unpacked.
chmod 744 Install/Install.sh # make sure that the install script can be executed.
./Install/Install.sh # execute the install script
#----------------------------------
